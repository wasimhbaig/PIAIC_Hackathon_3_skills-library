# Agents Documentation Generator

This skill generates AGENTS.md documentation files that describe the AI agents available in a codebase.

## Purpose

Automatically creates comprehensive documentation for AI agents, including their capabilities, tools, and usage examples.

## Usage

This skill analyzes agent configurations and generates structured documentation following a consistent format.

## Output

Generates an `AGENTS.md` file with:
- Agent descriptions
- Available tools and capabilities
- Usage examples
- Configuration details

## Example

See `example-output/AGENTS.md` for a sample generated file.

## Single Prompt Example

Project prompt:
> We are building an AI-powered learning platform using microservices, Kubernetes, and AI agents.

Result:
- A complete AGENTS.md is generated using this skill
